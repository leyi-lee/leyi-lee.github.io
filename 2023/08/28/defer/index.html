defer

知识点

  • 多个defer相当于放入一个栈，也就是先进后出，最先入栈的最后执行
  • deter 和 return 的执行顺序，return 先执行，defer 后执行
  • 函数的返回值初始化 func(i int) (t int) // t 会在函数起始会被初始化对应类型的零值并且作用于为整个函数
  • 函数返回值有名，在没有 defer 操作这个返回名和 return 没有区别，无返回值，名字相当于另一个变量
  • defer 遇到 panic 不捕获，输出玩 defer，panic 输出到标准错误 stderr
  • defer 捕获异常，正常执行
  • defer 中有 panic，会覆盖之前的 panic
  • defer 函数的形参包含子函数，因为需要压栈要知道参数是什么，所以要先把子参数执行完，再压栈
